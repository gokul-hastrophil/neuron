# Specification: Week 2 — The Brain: LLM Routing + Planning Engine

**Track ID:** brain-llm-routing_20260308
**Type:** Feature
**Created:** 2026-03-08

## Summary

Build Neuron's AI brain — tiered LLM routing (T0-T4), sensitivity gate, intent classification, ReAct planning loop, and a foreground brain service that orchestrates autonomous multi-step task execution across any Android app.

## Context

Week 1 ("The Nerve") delivered reliable UI read/write via AccessibilityService — UITreeReader, ActionExecutor, ScreenCapture, OverlayManager. Week 2 adds the intelligence layer: LLM clients that parse UI trees and produce actions, a router that selects the right model tier, and a plan-and-execute engine that loops until the task is done.

## User Story

As a **user**, I want to give natural language commands and have Neuron autonomously plan and execute multi-step tasks across any app, so I can control my phone hands-free.

As a **developer**, I want a well-structured LLM routing system with tiered models, so I can extend Neuron's intelligence while keeping sensitive data on-device.

## Acceptance Criteria

- [ ] `LLMRouter.route("send WhatsApp to Mom")` returns structured action plan JSON
- [ ] `SensitivityGate` correctly forces T4 for password fields, banking apps, health apps
- [ ] `PlanAndExecuteEngine` completes "Open Settings and turn on dark mode" in < 5 steps
- [ ] Full loop works: overlay input → brain service → executor → overlay feedback
- [ ] Confirmation gate pauses on irreversible actions (send, pay, delete)
- [ ] >70% success rate on 10-task benchmark across 5 apps
- [ ] All cloud LLM calls have timeouts and retry with exponential backoff
