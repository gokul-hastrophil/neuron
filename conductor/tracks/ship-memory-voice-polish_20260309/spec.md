## Context

Weeks 1-2 delivered the Nerve (AccessibilityService core) and Brain (LLM routing + planning engine). The 16-test integration benchmark showed 64% pass rate (9/14), with single-step tasks at 100% and multi-step tasks limited by LLM latency and 3 specific gaps:
1. No intent-based app resolution for OEM dialer/camera packages
2. Missing "press Enter" hint in system prompt for search bar scenarios
3. No NAVIGATE NOTIFICATIONS action mapping

This track completes the MVP: persistent memory so Neuron learns from past tasks, voice input for hands-free use, MCP server for developer extensibility, SDK skeleton for third-party tools, onboarding UX, and a signed beta APK.

## User Story

As a user, I want Neuron to remember my preferences, accept voice commands, and work reliably across all common phone tasks so I can control my phone hands-free from day one.

As a developer, I want an MCP server and SDK so I can extend Neuron with custom tools and integrate it into my workflows.
